## 🔢 Key KPIs

## 📊 Funnel Stages
| Stage | Count | Rate |
|-------|-------|------|
| Total Contacted | 41,188 | 100% |
| Engaged | 41,184 | 99.99% |
| Interested | 5,625 | 13.66% |
| Converted | 4,640 | 11.27% |

## 💡 Key Insights
- Biggest drop-off: Engaged → Interested (only 13.66% showed real interest)
- Only 11.27% of all contacts converted
- Job type and age group significantly impact conversion rate
- Targeted campaigns by job and age can improve conversion
